Dataset Information

Base SSL analysis script. This script logs information about the SSL/TLS handshaking and encryption establishment process.

Attribute Information

  • ts: Timestamp of request
  • uid: Connection unique id
  • id.orig_h: Originating endpoint’s IP address (AKA ORIG)
  • id.orig_p: Originating endpoint’s TCP/UDP port (or ICMP code)
  • id.resp_h: Responding endpoint’s IP address (AKA RESP)
  • id.resp_p: Responding endpoint’s TCP/UDP port (or ICMP code)
  • version: SSL version that the server offered
  • cipher: SSL cipher suite that the server chose
  • server_name: Value of the Server Name Indicator SSL extension
  • session_id: Session ID offered by the client for session resumption
  • subject: Subject of the X.509 cert offered by the server
  • issuer_subject: Signer Subject of the cert offered by the server
  • not_valid_before: NotValidBefore field value from the server cert
  • not_valid_after: NotValidAfter field value from the server cert
  • last_alert: Last alert that was seen during the connection
  • client_subject: Subject of the X.509 cert offered by the client
  • clnt_issuer_subject: Subject of the signer of the cert offered by the client
  • cer_hash: MD5 hash of the raw server certificate
  • validation_status: Certificate validation for this connection
